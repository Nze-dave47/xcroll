# xcroll

> a [fikfap](https://fikfap.com/) client

## pre-requisites

- [NodeJs](https://nodejs.org/en)

## installation

- clone the repository

```bash
git clone https://github.com/thedr4c0/xcroll.git
cd xcroll
```

- install dependencies

```bash
./setup.sh
```

## usage

- run the command in `xcroll` dir

```bash
npm start
```
